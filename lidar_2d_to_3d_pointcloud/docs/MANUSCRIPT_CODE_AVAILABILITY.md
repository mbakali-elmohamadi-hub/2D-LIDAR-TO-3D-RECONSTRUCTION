# Suggested Manuscript Code and Data Availability Statement

The source code, calibration and reconstruction scripts, Arduino firmware, visualization tool, and representative test datasets are provided in a dedicated reproducibility repository. The repository includes the complete 2D LiDAR to 3D point-cloud generation pipeline, covering LiDAR acquisition, Z-axis synchronization, Cartesian coordinate reconstruction, outlier filtering, and interactive visualization. The repository will be archived with a DOI through Zenodo upon public release.

Suggested wording:

> To support reproducibility and further research, the full implementation of the proposed 2D LiDAR-based 3D reconstruction framework is provided in an open-source repository. The repository contains the C++ LiDAR acquisition and 3D reconstruction code, Arduino Z-axis controller firmware, Python point-cloud filtering scripts, Java visualization interface, hardware parameters, and representative test datasets. The repository will be archived with a Zenodo DOI upon publication.
